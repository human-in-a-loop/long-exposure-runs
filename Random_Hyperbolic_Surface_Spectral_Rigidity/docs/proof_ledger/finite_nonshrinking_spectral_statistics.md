# Finite Non-Shrinking Spectral Statistics

## Setup

Let
\[
F(\Lambda)=\int_0^{\sqrt{\Lambda-1/4}}r\tanh(\pi r)\,dr,\qquad \Lambda\ge 1/4.
\]
The M2/M16 Weyl-law ledger gives, on the Kim--Tao high-probability event,
\[
N_{X_n}([1/4,\Lambda])
=(2g-2)nF(\Lambda)
+O_\epsilon\left(n^{1-\alpha_W}\Lambda^{1/2+\epsilon}\right).
\]

For a fixed interval \(I=[a,b]\), \(1/4\le a<b\), simultaneous endpoint subtraction gives
\[
N_{X_n}([a,b])
=(2g-2)n\left(F(b)-F(a)\right)
+O_\epsilon\left(n^{1-\alpha_W}b^{1/2+\epsilon}\right).
\]
This is the fixed-window theorem-level corollary.  Since \(F(b)-F(a)>0\) for every fixed positive-width interval, the main term is order \(n\), while the relative error is
\[
O_\epsilon\left(n^{-\alpha_W}
\frac{b^{1/2+\epsilon}}{(2g-2)(F(b)-F(a))}\right)
\]
with the displayed fixed energy factor absorbed into the constant for fixed \(a,b\).

## Centered Count

Equivalently,
\[
N_{X_n}([a,b])-(2g-2)n(F(b)-F(a))
=O_\epsilon\left(n^{1-\alpha_W}b^{1/2+\epsilon}\right).
\]
This is a deterministic high-probability centered bound, not a variance estimate and not a limiting distribution statement.

## Bulk, Edge, and High Energy

For \(a>1/4\),
\[
F'(\Lambda)=\frac12\tanh\left(\pi\sqrt{\Lambda-1/4}\right),
\]
so a fixed-width bulk interval has \(F(b)-F(a)\asymp b-a\) when it is away from the edge.  The relative error remains \(O(n^{-\alpha_W})\) up to the fixed energy constant.

At the spectral edge,
\[
F(1/4+\Delta)-F(1/4)
=\frac{\pi}{3}\Delta^{3/2}+O(\Delta^{5/2}).
\]
Thus fixed edge-adjacent windows still have order-\(n\) main term.  The edge only changes the fixed constant through \(\Delta^{3/2}\); it becomes exponent-relevant again only when \(\Delta\) shrinks with \(n\), which is outside M34.

At high fixed energy, \(F'(\Lambda)\to 1/2\), so the density stabilizes.  The endpoint error keeps the fixed factor \(b^{1/2+\epsilon}\).  This is harmless for fixed \(b\), but it is the reason M34 does not state a uniform-in-energy theorem.

## Rigidity Comparison

Rigidity gives reference locations \(\lambda_j\) satisfying
\[
|\lambda_j(X_n)-\lambda_j|\le C_\epsilon b^{1/2+\epsilon}n^{-\alpha_R}
=:\delta_R(b,n)
\]
for eigenvalues in the fixed energy range.  Therefore,
\[
\#\{j:\lambda_j(X_n)\in[a,b]\}
\le
\#\{j:\lambda_j\in[a-\delta_R,b+\delta_R]\},
\]
with the corresponding lower comparison using \([a+\delta_R,b-\delta_R]\) once this interval is nonempty.

For fixed \(b-a>0\), the expansion radius \(\delta_R(b,n)\to0\), so rigidity gives a valid comparison of random and reference counts.  It does not improve the count asymptotic beyond endpoint subtraction unless a sharper deterministic reference-count estimate is supplied.

## Scope Boundary

The rows generated by `scripts/analyze_finite_nonshrinking_spectral_statistics.py` intentionally include excluded \(\Delta=n^{-d}\) examples.  Those rows have relative exponent \(d-\alpha_W\) in the bulk and \(3d/2-\alpha_W\) at the edge, so they return to the M16-M25 threshold and support-obstruction branch.  M34 does not claim variance asymptotics, Poisson/Gaussian limits, level repulsion, local universality, or mean-spacing statistics.
